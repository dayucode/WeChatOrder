## 技术选型：
---
### 后台
- JDK8、MySQL、Spring-boot、Spring-data-jpa、Lombok、Freemarker、Bootstrap、Websocket
### 小程序端
- 微信小程序
### 小程序端
实现功能。
- 扫码点餐
- 菜品分类显示
- 模拟支付
- 评论系统

### 开发工具
IntelliJIDEA2019+Andriodstudio+微信开发工具
备注[因为使用习惯小程序端先在Andriodstudio开发APP端再转微信开发工具开发微信所需]
